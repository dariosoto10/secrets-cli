## SECRETS CLI

* `npm install` to install dependencies.
* `npx .` to create the database.
* `npx . secrets.db <command>` to execute a command.
* `npm run lint` to run the linter.

## Commands
* `users:list`     -
* `secrets:list`   receive the user
* `users:create`   receive the user and pass
* `secrets:get`    receive the user and name
* `secrets:delete` receive the user and name
* `secrets:create` receive the user, name and value
* `secrets:update` receive the user, name and value
